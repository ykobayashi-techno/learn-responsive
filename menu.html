メディアクエリ

メディアクエリを使って端末のサイズに合わせてCSSの値を切り替える

以下のように、@media screen and (条件)と書くことで画面幅を変えたときに切り替わりが起きる

.title {
    font-size: 1.8rem;
    margin: 0px;
    padding: 1em;
}

@media screen and (max-width: 767px) {
    .title {
        font-size: 1.2rem;
    }
}

条件は画面幅だけではなく、端末の向きなどの判定もある
